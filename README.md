# 💰 Employee Salary Prediction App

> 🚀 A Machine Learning-powered web application that predicts employee salaries based on experience, age, education level, and city using regression algorithms and an interactive Streamlit dashboard.

---

## 📖 Project Overview

This project estimates employee salaries using Machine Learning techniques. Users can provide employee details through an intuitive web interface and receive real-time salary predictions.

### 🎯 Objectives

* 🧹 Data Preprocessing
* ⚙️ Feature Engineering
* 🤖 Machine Learning Model Development
* 📈 Model Evaluation
* 🚀 Model Deployment
* 📊 Interactive Dashboard Creation

---

## ✨ Key Features

✔️ Real-Time Salary Prediction

✔️ Interactive Streamlit Web Interface

✔️ Employee Information Input Form

✔️ Machine Learning-Based Salary Estimation

✔️ Salary Insights & Visual Analytics

✔️ Dynamic Charts using Plotly

✔️ Production-Ready Deployment

## 🔄 Machine Learning Workflow

### 📥 Data Collection

Collected employee-related information such as:

* 👤 Age
* 💼 Experience
* 🎓 Education Level
* 🌆 City
* 🧠 Skills

### 🧹 Data Preprocessing

* Handling Missing Values
* Data Cleaning
* Feature Encoding
* Feature Selection

### 🏗️ Model Training

Trained and compared multiple regression models:

* Linear Regression
* Random Forest Regressor
* Bagging Regressor
* Gradient Boosting Regressor

### 📊 Model Evaluation

Performance evaluated using:

* R² Score
* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)

### 🚀 Deployment

The final model was deployed using Streamlit to provide a user-friendly web application.

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Manipavan-283/salary-prediction-app.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd salary-prediction-app
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

